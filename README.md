# EcoFuturo — Energias Renováveis no Brasil

Site desenvolvido como trabalho de APS (Atividades Práticas Supervisionadas) para a disciplina de Programação Web Responsiva da UNIP.

## Sobre o projeto

O site aborda o tema **Energias Renováveis no Brasil**, apresentando fontes de energia limpa, seu impacto ambiental e o potencial de transformação para o futuro do país. O tema foi escolhido pela relevância ambiental e pela conexão direta com soluções computacionais — desde sistemas de monitoramento de usinas até algoritmos de otimização energética.

## Páginas

| Página | Descrição |
|--------|-----------|
| `index.html` | Página inicial com introdução, tabela comparativa, vídeo e FAQ |
| `content.html` | Conteúdo aprofundado sobre solar, eólica, hidrelétrica e biomassa |
| `calculadora.html` | Ferramentas interativas: calculadora de CO₂, simulador solar e quiz |
| `contact.html` | Formulário de contato integrado com Formspree |
| `about.html` | Informações sobre o integrante e o trabalho |

## Tecnologias utilizadas

- HTML5 com tags semânticas
- CSS3 — Flexbox e Grid Layout
- JavaScript puro (ES5)
- Chart.js — gráfico de evolução da matriz elétrica brasileira
- Formspree — envio de formulário sem back-end
- GitHub Pages — hospedagem

## Funcionalidades

- Menu de navegação fixo (sticky) com indicação da página ativa
- Hero com chamada para ação
- Tabela comparativa entre fontes de energia renováveis e não renováveis
- Gráfico interativo da evolução da matriz elétrica (2015–2024)
- Accordion de perguntas frequentes
- Calculadora de pegada de carbono com barras de progresso e dicas personalizadas
- Simulador de viabilidade de energia solar residencial com dados reais de irradiação por estado (INPE/CRESESB)
- Quiz interativo com 5 perguntas, feedback imediato e pontuação
- Formulário de contato com validação client-side e envio assíncrono via Fetch API
- Toast de boas-vindas
- Layout responsivo para dispositivos móveis
